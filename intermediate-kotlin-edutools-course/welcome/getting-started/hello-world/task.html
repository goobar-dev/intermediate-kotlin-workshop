In this first section, you'll learn how to use the EduTools plugin to work through examples and exercises in support of your workshop.

Tracking Progress

The EduTools plugin will track your progress as you work through the examples. Each example should have a "call to action" button that says something like Run or Check.

Clicking this button will validate your code, and will update your progress once you've completed the task.

Some tasks, like this one, may have no code to write. In these cases, simply click Run when you're ready to proceed to the next task.
